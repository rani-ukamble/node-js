#Morgan
-- middleware commonly used with Express.js for logging HTTP requests. It provides a simple way to keep track of incoming requests to your server, making debugging and monitoring easier.
    Predefined formats:
    dev: Concise output colored by response status.
    combined: Standard Apache combined log output.
    common: Standard Apache common log output.
    short: Shorter than common.
    tiny: Minimal output.
